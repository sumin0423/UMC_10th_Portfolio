# UMC 10th Web Study

홍익대학교 UMC 10기 Web 파트 활동 기록 및 실습 과제를 정리한 저장소입니다.

## 학습 내용

- React
- TypeScript
- Vite
- React Router
- TanStack Query
- Redux Toolkit
- Zustand
- Vercel 배포

## 주차별 진행

- Week 0 ~ Week 10
- 실습(Practice)
- 미션(Mission)

## 목적

UMC Web 커리큘럼을 수행하며 프론트엔드 개발 역량을 향상시키고, React 기반 프로젝트 경험을 정리하기 위해 작성하였습니다.
